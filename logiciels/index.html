Prototypes logiciels

MapFIM

Le code source de MapFIM développée et un guide de démarrage pour son utilisation dans un environnement Hadoop/MapReduce sont disponibles en accès libre. Notons que cette application peut en particulier être utilisée pour rechercher certaines formes de sous-graphes fréquents dans une base de données de graphes (par exemple, des sous-graphes représentant des formes fréquentes de déplacements dans une métropole).

Site du logiciel: http://www.univ-orleans.fr/lifo/software/MapFIM

Context-driven Querying System for Graph Analysis

Context-driven Querying System for Graph Analysis propose (i) un langage de requête déclaratif offrant des outils pour l’analyse de graphes dans un environnement basé sur le contexte et (ii) un système d’interrogation qui filtre les réponses obtenues à partir d’une base de données en fonction d’un contexte donné.

Database Updating (marked nulls)

Database Updating (marked nulls) offre une plateforme où des faits et des règles sont insérés (pour simuler la base de données). Ensuite, des insertions et des suppressions peuvent être effectuées. La plateforme montre comment notre méthode utilise des algorithmes chase et core pour implémenter notre solution de mise à jour.

Site du logiciel: http://www.univ-orleans.fr/lifo/Members/Mirian.Halfeld/mi2-software.html

distGraph

distGraph est une bibliothèque d’algorithmes distribués de graphes pour architectures parallèles hétérogènes.

Site du logiciel: https://github.com/EmileCadorel/distGraph

DPcore

DPCore est un algorithme P2P de calcul d’une métrique d’influence dans les graphes qui est applicable à des graphes dynamiques et différentiellement privé. Le code repose sur {\it graph stream}, une bibliothèque pour faire de la visualisation de graphe dynamique. Il peut donc être lancé avec un rendu graphique montrant les modifications du graphe et le calcul en temps simulé.

Frequent Itemset Mining in Coq

FIM in Coq est une bibliothèque de formalisation d’algorithmes fonctionnels de recherche de motifs fréquents. Des techniques de transformation de programmes ont été appliquées pour obtenir des version optimisées à partir de versions non-optimisées.

Site du logiciel: https://sydpacc.gitbub.io

Formalization of a Big Graph API in Coq

Formalization of a Big Graph API in Coq est une bibliothèque de formalisation des primitives du DSL Fregel, un langage fonctionnel pour l’analys de grands graphes.
